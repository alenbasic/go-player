go-player

Introduction

go-player is a basic web interface to OMXPlayer. Compile it with go build go-player.go and run it with ./goplayer. Once it's running, go to http:\\ip-of-RPi:8080 for the fun to begin.

This is the go version of my other project, sto-player which is written in python and uses web.py and popen as external dependencies. If you compare the two programs with each other you'll notice there is a performance difference. On the RPi2 hardware, sto-player could sometimes take around 2 seconds to perform commands like starting or stopping a film. go-player however is pretty much instantaneous.

Requirements

go-player depends has no external dependencies and uses only the go standard library, so if you have go installed, that's all you need to compile the program and get started.

How to Use

You give it a list of directories and it takes care of the rest. The interface is pretty straight forward I believe, so I don't think there is much of a need of an explanation. Since there is always room for improvement, feel free to contact me regarding improvements or suggestions to the interface or the application itself.
