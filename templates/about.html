sto-player

Introduction

sto-player is a basic web interface to OMXPlayer. Start it up with python omxplayer.py and go to http:\\ip-of-RPi:8080 for the fun to begin.

Requirements

sto-player depends on web.py and popen. You can install both of those with either easy_install or pip (or manually if you really want), which may or may not be preinstalled on your RPi distribution of choice. Check the distrobution's documentation to find out.

How to Use

You give it a list of directories, it takes care of the rest. The interface is pretty straight forward I believe, so I don't think there is much of a need of an explanation. Since there is always room for improvement, feel free to contact me regarding improvements or suggestions to the interface or the application itself.
